## 自动注册-配置说明


#### 1.接收Cursor验证码的临时邮箱

程序需要一个临时邮箱来接收 Cursor 发送的验证码。您可以前往 [tempmail.plus](https://tempmail.plus) 获取免费的临时邮箱和对应的PIN码。
